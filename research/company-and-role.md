## Company Background

### Concentrate AI

- **Product**: Unified LLM API gateway — one API for 50+ models across 10+ providers
- **Tagline**: "Deploy & Manage AI at Scale. One API for all LLMs with zero markup, tokens back, and private by design."
- **Stage**: Seed, well-funded, stealth
- **HQ**: NYC (130 E 59th St, 17th floor) + Wilmington, DE
- **Founders**: Zach Moskow + Anna (co-founder)
- **Competitors**: LiteLLM (open-source, most similar), Portkey, Requesty, Kong AI Gateway

### Business Model

- No platform fees — pay only for token usage
- Bulk rate negotiation → Token Credits passed to customers
- Free: analytics, logs, spend management, alerts, guardrails, SSO, RBAC
- Revenue: margin on token pricing + enterprise features

### Supported Providers (10+)

OpenAI, Anthropic, Google Vertex AI, AWS Bedrock, Azure OpenAI, xAI, Cohere, Mistral, Cloudflare, Hugging Face, DeepSeek, Qwen

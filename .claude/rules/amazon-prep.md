# Amazon DS Interview Prep Rules

Amazon Sr. Data Scientist interview prep guide for Books Ads team.

## Quick Reference

```bash
cd amazon_ds_interview_prep_2026/latex_version
make digital      # Build digital PDF
make bw           # Build B&W print variant
make pilot        # Quick test build (single pass)
make clean        # Remove generated files
```

## Build Requirements

```bash
sudo apt install texlive-full latexmk
pip install Pygments
```
